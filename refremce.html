About This Project

Local Eats is a student-led web development project created as part of an academic assignment at Kent Institute Australia for the Web Development unit. The aim of the project is to design and build a fully functional website that highlights and promotes local food vendors, helping to increase the visibility of small and independent eateries within the community. Through this project, students apply theoretical knowledge gained in class, including concepts of HTML, CSS, JavaScript, and responsive web design, to create a practical, real-world application. Local Eats focuses on providing an intuitive user experience by implementing modern design standards and mobile-friendly layouts using tools such as Bootstrap 4. The project also demonstrates an understanding of digital branding, usability, and accessibility, reflecting current trends in the food and hospitality industry. This assignment not only tests technical development skills but also encourages creativity, problem-solving, and the ability to integrate feedback and improvements throughout the design process.

Resources and Technologies Used

The layout and responsiveness were achieved using Bootstrap 4, which provided a solid grid system and pre-built components (Bootstrap, 2024). HTML, CSS, and JavaScript elements were implemented with the help of tutorials and examples from W3Schools (W3Schools, 2024).

Canva was used for designing banners and visual elements to enhance the user experience (Canva, 2024). TikTok served as a source of creative inspiration, particularly in identifying UI trends common in food-related content (TikTok, 2024). Throughout the development process, syntax checking and language polishing were supported using ChatGPT (OpenAI, 2024). And All pages were validated using the W3C Validator to ensure compliance with web standards and eliminate HTML/CSS errors (W3C, 2024).
